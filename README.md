## Dev

```sh
# Project Setup
pnpm install
# Compile and Hot-Reload for Development
pnpm dev
# Type-Check, Compile and Minify for Production
pnpm build
# Lint with ESLint
pnpm lint
# Format with Pretter
pnpm format
```
